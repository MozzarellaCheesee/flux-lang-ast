# Flux Programming Language

Flux — это экспериментальный статически типизированный язык программирования с синтаксисом в стиле C/Rust. Язык компилируется в нативные объектные файлы и затем линкуется в исполняемый бинарник.

---

## Использование

Для работы компилятора у вас должен быть установлен один из следующих компиляторов:
- gcc
- clang
- msvc

---

## Возможности

- Статическая типизация
- Базовые числовые типы — знаковые и беззнаковые целые, числа с плавающей точкой
- Строковые типы `&str` (как в Rust) и `string`
- Управляющие конструкции: `if / else`, `while`, `for`, `match`
- Структуры (`struct`), классы (`class`), методы через `impl`
- Операции с указателями и ссылками (`&`, `*`, `.`)
- Встроенные функции вывода (`println`, `print`)

## Базовый пример

```flux
fnc main() -> int32 {
    println("Hello, Flux!");

    let a: int32 = 6;

    match a {
        4 => { println("four"); },
        6 => { println("six"); },
        _ => { println("no match"); }
    }

    return 0;
}
```

## Объявление переменных

Ключевое слово `let`:

```flux
fnc main() -> int32 {
    let x: int32 = 10;
    let y: float  = 3.14;
    let s: &str   = "hello";
    let flag: bool = true;
    return 0;
}
```

Тип указывается явно через `:`. Вывод типа из инициализатора поддерживается, если тип не указан.

---

## Функции

Функции объявляются с помощью ключевого слова `fnc`:

```flux
fnc add(a: int32, b: int32) -> int32 {
    return a + b;
}

fnc greet(name: &str) {
    println(name);
}

fnc main() -> int32 {
    let result: int32 = add(2, 3);
    greet("Flux");
    return 0;
}
```

- Возвращаемый тип указывается после `->`.
- Функция без возвращаемого значения объявляется без `->`.
- Модификатор `pub` делает функцию публичной (External linkage):

```flux
pub fnc exported() -> int32 {
    return 42;
}
```

## Управляющие конструкции

### if / else

```flux
fnc main() -> int32 {
    let x: int32 = 5;

    if x < 10 {
        println("small");
    } else {
        println("big");
    }

    return 0;
}
```

---

### while

```flux
fnc main() -> int32 {
    let i: int32 = 0;

    while i < 5 {
        println("loop");
        i++;
    }

    return 0;
}
```

---

### for

```flux
fnc main() -> int32 {
    for let i: int32 = 0; i < 10; i++ {
        println("tick");
    }
    return 0;
}
```

В теле цикла можно использовать `break` и `continue`:

```flux
fnc main() -> int32 {
    let i: int32 = 0;
    while true {
        if i == 3 { continue; }
        if i == 7 { break; }
        i++;
    }
    return 0;
}
```

---

### match

Конструкция `match` позволяет сопоставлять значение с набором шаблонов:

- Числовые литералы
- Строковые литералы
- Подстановочный шаблон `_` (wildcard — совпадает с любым значением)

```flux
fnc main() -> int32 {
    let a: int32 = 6;

    match a {
        4 => { println("4"); },
        6 => { println("6"); },
        _ => { println("no"); }
    }

    return 0;
}
```

Пример с `&str`:

```flux
fnc main() -> int32 {
    let s: &str = "hello";

    match s {
        "hello" => { println("greeting"); },
        "bye"   => { println("farewell"); },
        _       => { println("unknown"); }
    }

    return 0;
}
```

## Структуры, классы и методы

### struct

```flux
struct Point {
    pub x: int32
    pub y: int32
}

fnc main() -> int32 {
    let p: Point = Point{ x: 10, y: 20 };
    return 0;
}
```

- `pub` помечает поле как публичное.

### impl — реализация методов

```flux
struct Counter {
    pub value: int32
}

impl Counter {
    fnc increment(self: &Counter) {
        self.value++;
    }

    fnc get(self: &Counter) -> int32 {
        return self.value;
    }
}

fnc main() -> int32 {
    let c: Counter = Counter{ value: 0 };
    c.increment();
    return 0;
}
```

### class — структура + методы в одном объявлении

```flux
class Rect {
    pub w: int32
    pub h: int32

    pub fnc area(self: &Rect) -> int32 {
        return self.w * self.h;
    }
}

fnc main() -> int32 {
    let r: Rect = Rect{ w: 5, h: 3 };
    let a: int32 = r.area();
    return 0;
}
```

## Указатели и ссылки

```flux
struct Data {
    pub value: int32
}

fnc increment(d: &Data) {
    d.value += 1;
}

fnc main() -> int32 {
    let obj: Data = Data{ value: 0 };
    increment(&obj);
    return 0;
}
```

- `&x` — взятие адреса (получение ссылки/указателя).
- `*ptr` — разыменование указателя.
- `.` — доступ к полям и методам.

## Препроцессор

Директива `#import` включает содержимое другого файла до парсинга:

```flux
#import "std.flx"
#import "math.flx"

fnc main() -> int32 {
    println("hello");
    return 0;
}
```

## Встроенные функции

| Функция                 | Описание                              |
|-------------------------|---------------------------------------|
| `println(&str)`         | Печатает строку + перевод строки      |
| `print(&str)`           | Печатает строку без перевода строки   |
| `println_int(int64)`    | Печатает целое + перевод строки       |
| `println_float(double)` | Печатает float + перевод строки       |
| `println_bool(bool)`    | Печатает `true` / `false` + `\n`      |

## Ключевые слова

| Ключевое слово | Описание                                         |
|----------------|--------------------------------------------------|
| `fnc`          | Объявление функции                               |
| `let`          | Объявление переменной                            |
| `return`       | Возврат значения из функции                      |
| `if`           | Условный оператор                                |
| `else`         | Ветка условного оператора                        |
| `while`        | Цикл while                                       |
| `for`          | Цикл for                                         |
| `continue`     | Переход к следующей итерации цикла               |

| `true`         | Булев литерал истина                             |
| `false`        | Булев литерал ложь                               |
| `pub`          | Модификатор публичности метода или поля          |
| `struct`       | Объявление структуры                             |
| `impl`         | Реализация методов для структуры                 |
| `class`        | Объявление класса (struct + impl)                |
| `_`            | Подстановочный паттерн в `match`                 |

## Архитектура

```
source.flx
    │
    ▼ Lexer                  Токенизация исходного кода
  tokens
    │
    ▼ Preprocessor           Раскрытие #import
  expanded tokens
    │
    ▼ Parser                 Построение AST
  AST (ProgramNode)
    │
    ▼ SemanticAnalyzer       Проверка типов, разрешение имён
  проверенный AST
```
